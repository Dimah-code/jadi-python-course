## Positive or Negative Exercise

Write a function called "is_positive" that takes a number and returns True if the number is positive or zero, and False otherwise.

### Input:

- An integer (int).

### Output:

- True if positive or zero.
- False if negative.

### Sample Input 1:
```bash
 5
```

### Expected Output 1:
```bash
 True
```

### Sample Input 2:
```bash
 -3
```

### Expected Output 2:
```bash
False
```
